# Mac Desktop Setup

## Prerequisites 

1. Install Brew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

2. Install Ansible

```bash
brew install ansible
```

3. Run installers

### General setup
```bash
ansible-pull -U https://github.com/dave-borg/mac-desktop-setup.git
```

### Utilities
